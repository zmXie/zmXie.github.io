    Vue单页应用结构

    一个单页面应用就是一个 Vue 实例，通过 new Vue(…)挂载到 dom 元素上。 每一个页面都是一个 vueComponent 实例（vueComponent 继承自 Vue），通过 Router 建立页面组件与 hash 的映射表（hash 是 url 中#号后面的值）。 当改变 url 中的 hash 值时，会通过路由映射表找到对应的组件，挂载到 Vue 实例下，然后更新虚拟 dom，最后渲染到页面上。
